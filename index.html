Capstone / Medical Device

Pulmonary Unit for Live Sensing & Evaluation (P.U.L.S.E.)

Designed an attachment for a bag valve mask that integrates sensing and embedded processing to monitor respiratory parameters. I also created a companion application to visualize live sensor data, present key ventilation metrics, and make the system easier to monitor through a clear user-facing interface. Together, the hardware and application improve visibility into ventilation performance through continuous feedback, embedded data acquisition, and accessible display interfaces.

Controls / Modeling

PID Ball Balancing Platform

Developed a real-time ball balancing platform that combines resistive touchscreen position sensing, PID-based closed-loop control, and servo-driven actuation to keep a ball centered on the surface. The project involved system modeling, controller tuning, and prototype validation on a Raspberry Pi-based testbed, demonstrating stable feedback control in a dynamic system.

Automation / Robotics

Automated Stack Sorting Robot

Designed and assembled an autonomous stack sorting robot that uses computer vision to identify colored blocks, align with the target, and transport each block to a designated stacking location. The system integrates a Raspberry Pi, custom 3D-printed elevator and claw mechanisms, and stepper and servo motor control to coordinate sensing, gripping, lifting, and placement in a complete workflow.

Computer Vision Raspberry Pi Stepper & Servo Control 3D Printing
Dynamics / Mechatronics

Hanging Crane with Double Pendulum System

Developed a hanging crane system with pendulum dynamics to study motion behavior, load swing response, and control considerations in an underactuated mechanical system. The project also explores double pendulum behavior, where small differences in initial conditions can produce highly unpredictable and chaotic motion. This work highlights dynamic modeling, system analysis, and the challenges of understanding and controlling complex nonlinear systems in system design.

Embedded / Microcontroller

LED Reaction Game with UART Scoring

Programmed a microcontroller-based game where users respond to LED prompts with button inputs. Reaction timing was converted into points and transmitted results over UART for mobile app interaction. The project involved coordinating timing logic, user input handling, and serial communication to create a responsive embedded game with real-time scoring feedback.
